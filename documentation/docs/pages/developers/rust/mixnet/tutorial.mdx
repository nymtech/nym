# Tutorial: Send Your First Private Message

import { Callout } from 'nextra/components'

Build a program that connects to the Nym Mixnet, sends a message to itself, receives it, and replies anonymously using SURBs.

<Callout type="info">
This tutorial assumes you've already added `nym-sdk` and `tokio` to your project. See [Installation](../importing) if not.
</Callout>

## Connect and send

```rust
use nym_sdk::mixnet::{self, MixnetMessageSender};

#[tokio::main]
async fn main() {
    let mut client = mixnet::MixnetClient::connect_new().await.unwrap();
    let our_address = client.nym_address();
    println!("Connected: {our_address}");

    // The message is Sphinx-encrypted and mixed across 5 nodes.
    client
        .send_plain_message(*our_address, "hello from the mixnet!")
        .await
        .unwrap();

    println!("Sent — waiting for arrival...");
```

`send_plain_message` only blocks until the message is queued — encryption and mixing happen in background tasks. The round trip takes a few seconds.

## Receive

```rust
    // Filter empty messages — these are SURB replenishment requests.
    let message = loop {
        if let Some(msgs) = client.wait_for_messages().await {
            if let Some(msg) = msgs.into_iter().find(|m| !m.message.is_empty()) {
                break msg;
            }
        }
    };

    println!("Received: {}", String::from_utf8_lossy(&message.message));
```

## Reply anonymously

Every message includes a `sender_tag` — an opaque token that lets you reply **without knowing the sender's address**:

```rust
    let sender_tag = message.sender_tag.expect("should have sender tag");
    client.send_reply(sender_tag, "hello back, anonymously!").await.unwrap();

    // Receive the reply
    let reply = loop {
        if let Some(msgs) = client.wait_for_messages().await {
            if let Some(msg) = msgs.into_iter().find(|m| !m.message.is_empty()) {
                break msg;
            }
        }
    };

    println!("Reply: {}", String::from_utf8_lossy(&reply.message));

    client.disconnect().await;
}
```

```sh
cargo run
```
