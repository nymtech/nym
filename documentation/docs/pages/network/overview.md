# Overview

Start here if you want to understand what the Nym Network actually does and why it was built.

[The Privacy Problem](/network/overview/privacy-problem) explains the metadata problem and the adversaries Nym is designed against. [Two Modes](/network/overview/two-modes) covers how dVPN and Mixnet mode work differently under the hood. [Choosing a Mode](/network/overview/choosing-a-mode) helps you pick which one to use. [Network Components](/network/overview/network-components) breaks down the infrastructure, and [Comparisons](/network/overview/comparisons) puts Nym side by side with VPNs, Tor, and I2P.
