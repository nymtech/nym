# Nym Tauri Wallet 

A Rust and Tauri desktop wallet implementation.  

## Installation prerequisites 

* `Yarn`
* `NodeJS >= v16.8.0`
* `Rust & cargo >= v1.51` 

## Installation & usage

* `yarn install`
* `yarn dev`
