# Nym Network Explorer - Development Docs

## Getting started

You will need:

- NodeJS
- `nvm`

We use the following:

- Typescript
- `eslint`
- `webpack`
- `jest`
- `react-material-ui`
- `react` 17

## Development mode

Run the following:

```
npm install
npm run start
```

A development server with hot reloading will be running on http://localhost:3000.
